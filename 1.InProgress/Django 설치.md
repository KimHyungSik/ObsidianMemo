파이썬의 가상환경을 사용하는 이유
프로젝트중 의존성을 가지는 라이브러리 업데이트 시 문제 발생을 낮추기위하여 프로젝트 개별의 가상환경을 설정하여 사용한다.
가상환경 설치 방법
```shell
python -m venv munang_dashboard
cd munang_dashboard
```
가상환경 활성환 방법
```null
cd bin
source ./activate
```

가상환경 비활성화 : /deactivate
