### 주제 : #Multipart
----
### 정리
Multipart란 HTTP프로토콜로 Body에 파일을 전달할때 사용하는 Content-type입니다. 
-   보통 Multipart 요청을 보낼 때 HTTP 메소드는 `POST`를 사용한다.
-   `Content-Type`이 `multipart/form-data`이다.
