### 주제 : #Kotlin #Serializtion

### 정리
기존에는 Java의 Serialization또는 , Gson, Moshi 등을 사용하여 직렬화 또는 역직렬화를 사용해 왔다. 하지만 이모두 Java 언어로 개발되어있어 Kotlin환경에서는 완벽하게 호환되지 않는다. 예를 들어 Kotlin data class를 사용할 때 기본값을 정해 줄 수 있ㄷ.
